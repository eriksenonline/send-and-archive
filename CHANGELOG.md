# Changelog

All notable changes to this project will be documented in this file.

## [1.0.2] - 2026-03-03

### Maintenance
- Cut a new release for validation testing.
- Keep automatic `.xpi` packaging flow via `./scripts/package.sh`.

## [1.0.1] - 2026-03-03

### Documentation
- Added MIT open-source license.
- Added clear README disclaimer that the extension is for personal use, unsupported, and updated only when personally needed.
- Clarified that community contributions are welcome.

## [1.0.0] - 2026-03-03

### Features
- Added Thunderbird MailExtension with compose toolbar action for **Send & Archive**.
- Added `Ctrl+Shift+Enter` command for send-and-archive.
- Implemented send-then-archive flow so archive only runs after confirmed send.
- Added English and Norwegian Bokmål localizations.
- Added base README documentation.
